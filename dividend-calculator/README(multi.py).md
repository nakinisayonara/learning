# 多市場股息分析工具(multi.py)

## 簡介
此工具能自動抓取股票的股價、股息紀錄，幫助比較不同市場的收息能力。數據源自yfinance的股票和貨幣匯率即時數據。可以輸入**多個股票**（可跨地區），最終轉換成**港幣**去比較。另外還有視覺化圖表幫助比較和分析。

## 使用方法(缺少套件會導致無法運行)
Python 版本需求 (>=3.8)」或「建議使用虛擬環境」
安裝必要套件：
pip install -r requirements.txt
或
pip install yfinance pandas matplotlib numpy
執行：
python multi.py
進行輸入時保持輸入法為英文輸入法/半形文字

## 功能
- **跨市場支援**：台股、美股、港股、日股、歐股等  
- **即時匯率換算**：統一換算成港幣 HKD  
- **匯率敏感度分析**：模擬 ±5% 匯率波動對股息收入的影響  
- **股息明細表**：逐次派息紀錄（本幣 & 港幣）  
- **年度股息比較圖表**：直觀比較不同股票的年度股息收入  
- **36 個月股息趨勢圖**：檢視派息穩定度與成長性  
- **股息穩定度指標**：計算近 36 個月股息標準差與成長率  

## 輸出結果
總表： 
    年度/月度股息收入（自動轉換港幣）
        - 股息率
        - 真實現金回報數據
    匯率敏感度分析
        - +-5%匯率波動
    股息穩定度指標
        - 標準差低，股息穩定
        - 成長率高，股息逐年增加
明細表：
    逐次派息紀錄（本幣 & 港幣）
        - 近一年的派息記錄
        - 年度/半年/季度/月度派息
圖表：
    年度股息比較
        - 不同顔色的重叠柱狀圖比較(+—5%匯率波動比較)
        - 本幣為港幣並沒有匯率波動比較
    36 個月股息趨勢
