# 股息計算器(single.py)

## 簡介
此工具能自動抓取股票的股價、股息紀錄，幫助比較不同市場的收息能力。數據源自yfinance的股票和貨幣匯率即時數據。
輸入**單個股票**查詢股息數據和視覺化圖表。

## 使用方法(缺少套件會導致無法運行)
Python 版本需求 (>=3.8)」或「建議使用虛擬環境」
安裝必要套件：
pip install -r requirements.txt
或
pip install yfinance pandas matplotlib numpy streamlit
執行：
python multi.py
進行輸入時保持輸入法為英文輸入法/半形文字

## 功能
- **跨市場支援**：台股、美股、港股、日股、歐股等  
- **股息紀錄**：約一年派息紀錄（本幣）  
- **年度股息與月度股息**：計算年股息和平均月股息收入

## 輸出結果
股息紀錄：
    有約一年間的股息紀錄
計算總結：
    每股股息，股息率，年股息/平均每月股息收入...
